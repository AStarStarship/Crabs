title: Meta Types
layout: page
---
Meta Types are ASCII Data Types required to implement the specification but are not actual data types because they are subclasses of data types with only meta additions. They are unordered and thus do not have indexes.


|  Type   |    Name      | Description                                |
|:-------:|:------------:|:-------------------------------------------|
|   CHL   | Char Largest | The largest char type, either CHA or CHC.  |
|   CHS   | Char String  | The character type of the string.          |
|   ISN   |  signed int  | C++ int type.                              |
|   IUN   | unsigned int | C++ unsigned int type.                     |
