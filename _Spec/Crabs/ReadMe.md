# [Crabs Specification](./)

The **Chinese Room Abstract Stack Machine** (**Crabs**) is an abstract machine capable of running *Group Automata* using the SCRIPT Protocol and Crabs. Crabs is the standardized Crabs API for interacting with a Chinese Room with a text terminal over the Script. Script is designed to run over a standard serial connection that allows users to hack on devices in real-time with a PC keyboard, app GUI, or other serial B-Stream so Script makes use of ASCII C0 Codes and VK Keyboard Codes so you can do alt, shift, or control keys on the keyboard to do stuff like copy and paste and Script Macros.

## Requirements

1. The Chinese Room Abstract Stack Machine (Crabs) must be capable of running Group Automata using the SCRIPT Protocol and Crabs.
2. Crabs must provide a standardized Crabs API for interacting with a Chinese Room with a text terminal over the Script.
3. Script must run over a standard serial connection that allows users to hack on devices in real-time.
4. Script must support ASCII C0 Codes and VK Keyboard Codes for keyboard input.
5. Script must support alt, shift, or control keys for copy, paste, and Script Macros.
6. Crabs must support multiple terminal interfaces including PC keyboard, app GUI, and other serial B-Streams.
7. The machine must be designed to minimize RAM, ROM, and complexity while maximizing code reuse.
8. The machine must be portable to almost any microprocessor and pen and paper computer.
9. The machine must support international language support.
10. The machine must operate in a safe manner with proper error handling and recovery mechanisms.
11. The machine must be useful in multiple contexts including embedded systems, IoT devices, and educational environments.
